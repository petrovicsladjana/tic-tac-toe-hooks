# Tic Tac Toe Minimax

Projekt predstavlja demonstracijo uporabe Git Hooks na Python projektu.

## Uporabljeni Git Hooki

- pre-commit
- commit-msg
- pre-push

## Statična analiza

- flake8
- pylint
- bandit

## Dinamična analiza

- unittest
- coverage

## Struktura projekta

- py_version/minimax.py
- py_version/test_tictactoe.py
- web_version/
- preview/

## Commit message pravilo

Uporabljen je Conventional Commits format:

<type>: <description>

Primer:
test: add minimax unit tests
